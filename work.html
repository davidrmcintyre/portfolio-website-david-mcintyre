CityCrawlers Meet APP





The project uses OAuth from google to verify users and allow the app to reach the Google Calender API.


It then displays the main page of the app that shows events taking place over several cities.


It is then possible to select an individual city, and set the number of events you wish to see.


Finally, the events themselves are displayed.

In this project I used React to create a "Meet" app using an external API. I also wrote unit and integration tests for the app following both TDD and BDD.
